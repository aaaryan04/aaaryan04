## 🧭 Featured Project — Hacktrek WebCrawler

</div>

**An open-source reconnaissance dashboard for authorized website security assessments.**

Consolidates the early-stage recon steps a pentester normally runs across separate CLI tools — headers, DNS, TLS, WAF, and technology fingerprinting — into a single dashboard with exportable results.
